# Angkor.dev

## About Us
Angkor.dev is a forward-thinking technology company focused on building scalable, high-quality product-based solutions. Our mission is to empower businesses and individuals through innovative digital products that solve real-world problems and deliver exceptional user experiences.

## What We Do
We specialize in end-to-end product development, from ideation and design to deployment and ongoing support. Our flagship products, such as [tovtam.com](https://tovtam.com), showcase our commitment to robust architecture, intuitive interfaces, and seamless performance.

## Tech Stack

- **Backend:**
  - **Laravel (PHP):** Modern, secure, and maintainable framework for rapid web application development.
  - **Spring Boot (Java):** Enterprise-grade APIs and microservices for scalability and reliability.
  - **NestJS (Node.js):** Progressive Node.js framework for efficient, scalable server-side applications.
  - **Phoenix (Elixir):** High-performance, real-time web applications leveraging the power of Elixir.

- **Frontend:**
  - **Next.js:** React-based framework for server-side rendering and static site generation.
  - **Nuxt.js:** Vue.js framework for universal applications and static site generation.
  - **React Native:** Cross-platform mobile app development for iOS and Android.

- **DevOps & Cloud:**
  - **Kubernetes:** Container orchestration for scalable, resilient, and automated deployments.
  - CI/CD pipelines, containerization, and cloud-native deployments ensure our products are robust, secure, and easy to scale.
